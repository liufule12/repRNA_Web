Physicochemical indices format

The format of the user-defined physicochemical index files: each index should be represented in three lines.
The first line must start with a greater-than symbol (">") in the first column. The words right after the ">" symbol in the single initial line are optional and only used for the purpose of identification and description of the index.
The second line lists the names of the sequence compositions (i.e. amino acids, nucleotides, dinucleotides, or trinucleotides, etc), which should be sorted in the alphabet order, such as 'A' 'C' ... 'AA' 'AC'.
The corresponding values of these sequence compositions are listed in the third line, which are separated by "tab".
All the elements in this line should be separated by "tab".
A examples are shown in RNA_indices.txt.
